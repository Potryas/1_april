Запуск в один клик:

1. Дважды нажми [start-choir.cmd](D:\Codex_projects\Horses\start-choir.cmd)
2. Откроется отдельное окно сервера
3. Игра откроется в браузере по адресу http://localhost:4173

Если не запустилось:
- проверь, установлен ли Node.js
- если браузер не открылся сам, открой http://localhost:4173 вручную
